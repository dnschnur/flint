# Flint

Flint is a retirement simulator that uses an approach inspired by the excellent ficalc.app.

It allows you to model your assets, income, and budget, then uses the historical performance of the
S&P 500 to run a Monte Carlo simulation of how you would perform financially if you retired in a
particular year.

Unlike FI Calc, which models assets as a simple mix of stock and bonds, Flint allows more granular
categorization of assets, including 401K and IRA accounts (Roth and non-Roth), HSA accounts, and
college savings 529 accounts.

Spending similarly supports categories including housing, medical care, education, and food, each
with its own inflation rate, to account for the fact that standard measures of overall inflation
poorly capture household expenses.

It supports improved customization in other areas as well, such as post-retirement income.
